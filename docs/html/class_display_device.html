Detailed Description

This is the abstract class to handle all generic behavior of physical graphic display devices (e.g. OLED screen SSD1306). It derived from Framebuffer. This allows to draw graphics directly into the display framebuffer thanks to Framebuffet class graphic primitives indepently from any kind of widget.
