Detailed Description

This is the abstract class to handle all generic behavior of physical display devices (e.g. OLED screen SSD1306). It derived from Framebuffer. This allows to draw text and graphics directly into the display framebuffer thanks to Framebuffet class text and graphic primitives indepently from any kind of widget.

This is the Model abstract class of Model_View_Control design pattern.

It handles change_flag, a semaphore used to indicate that a screen draw_refresh is required.

The controller or any other entities that modify the model must set the change_flag and the widget in charge of its screen representation must clear the change_flag
