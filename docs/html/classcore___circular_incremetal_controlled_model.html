Detailed Description

A model that holds an integer value that can be incremented or decremented within a defined range in a circular manner. For example, incrementing beyond the max value wraps around to the min value, and decrementing below the min value wraps around to the max value. This is particularly useful for representing circular quantities such as angles (0 to 360 degrees). In such case 360 = 0.
